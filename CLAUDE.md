# AI Dev Capstone

## About
Capstone project for the AI-assisted development track. Built with the help of Claude Code as a pair-programming assistant.

## Stack
- Language: JavaScript / TypeScript
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Version control: Git, GitHub

## Conventions
- Commit messages follow Conventional Commits (e.g. `feat:`, `fix:`, `docs:`, `chore:`, `refactor:`)
- Use ESM imports (`import`/`export`), not `require()`
- Keep functions small and single-purpose
- Write clear variable and function names — avoid abbreviations
- Add comments only where logic isn't self-explanatory

## Project Structure
- `/src` — application source code
- `/tests` — test files
- `README.md` — project overview and setup instructions

## Commands
- `npm install` — install dependencies
- `npm run dev` — start development server
- `npm test` — run tests

## Notes for Claude Code
- Always explain the plan before making multi-file changes
- Prefer readable code over clever one-liners
- Flag any assumptions made when requirements are unclear
